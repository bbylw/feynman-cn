功能概览

从提问到发表，
覆盖完整研究链路

自然语言输入或使用斜杠命令，Feynman 自动编排代理完成研究任务。

深度调研

多代理并行搜索论文与网络资源，自动综合分析与交叉验证。

/deepresearch <topic>

文献综述

原始文献搜索，自动梳理领域共识、观点分歧和开放问题。

/lit <topic>

同行评审

模拟同行评审流程，按严重程度分级并生成修订计划。

/review <artifact>

代码审计

对比论文声明与公开代码库，发现不匹配之处。

/audit <item>

实验复现

在本地或云端 GPU 上复现论文实验，验证研究结论。

/replicate <paper>

草稿撰写

基于研究发现自动撰写论文式结构化草稿。

/draft <topic>

命令速查

10 个命令，
覆盖全部研究工作流

每个命令都是一个完整的研究工作流。自然语言输入同样有效 —— 斜杠命令只是快捷方式。

命令 功能
/deepresearch 深度多源调研
/lit 论文搜索和原始文献综述
/review 模拟同行评审，含严重程度和修订计划
/audit 论文与代码库不匹配审计
/replicate 在本地或云端 GPU 复现实验
/compare 来源对比矩阵
/draft 基于研究发现撰写论文式草稿
/autoresearch 自主实验循环
/watch 持续研究监控
/outputs 浏览所有研究成果

工作原理

构建在成熟的开源基础之上

基于 Pi 构建代理运行时，使用 alphaXiv 进行论文搜索和分析，通过 CLI 工具实现计算和执行。功能以 Pi 技能形式交付 —— Markdown 指令文件在启动时同步到 ~/.feynman/agent/skills/。

快速开始

一行命令，立即安装

安装程序下载独立原生包，包含自带的 Node.js 运行时。

macOS / Linux
$ curl -fsSL https://feynman.is/install | bash
Windows (PowerShell)
PS> irm https://feynman.is/install.ps1 | iex

仅安装研究技能

如果只需要研究技能而不需要完整终端应用，可以单独安装技能包。

$ curl -fsSL https://feynman.is/install-skills | bash
PS> irm https://feynman.is/install-skills.ps1 | iex

仓库本地安装请添加 --repo 参数（Linux/macOS）或 -Scope Repo 参数（Windows）。

支持本地模型

运行 feynman setup 可配置 LM Studio、LiteLLM Proxy、Ollama 或 vLLM 等本地模型服务。
